## Usage

```html
<h1 class="display-1">Breaking News</h1>
<p class="body-md">Standard Fließtext...</p>
```

```css
.custom-headline {
  font-family: var(--font-family-gotham-xnarrow);
  font-weight: var(--font-weight-bold);
}
```
