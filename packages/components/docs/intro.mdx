# BILD Design System

Welcome to the **BILD Design System** - a comprehensive design token system for consistent brand experiences across platforms.

<div className="hint-box">
  <strong>Developer Tip:</strong> Use the <strong>toolbar controls above</strong> to switch between brands, themes, and density modes. All documentation adapts automatically.
</div>

---

## 4-Layer Token Architecture

<table className="intro-table">
  <thead>
    <tr>
      <th>Layer</th>
      <th>Name</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>0</td>
      <td>Primitives</td>
      <td>Raw values (colors, sizes, fonts) - no modes</td>
    </tr>
    <tr>
      <td>1</td>
      <td>Mapping</td>
      <td>Brand-specific mapping (BILD, SportBILD, Advertorial)</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Semantic</td>
      <td>Meaning-based tokens (ColorMode, BreakpointMode, Density)</td>
    </tr>
    <tr>
      <td>3</td>
      <td>Components</td>
      <td>Component-specific tokens (~55 components)</td>
    </tr>
  </tbody>
</table>

## Dual-Axis Architecture
