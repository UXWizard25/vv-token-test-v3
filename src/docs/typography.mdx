# Typography

Das BILD Design System verwendet **Gotham** als primäre Schriftfamilie.

## Font Families

| Name | CSS Variable | Fallback |
|------|--------------|----------|
| Gotham XNarrow | --font-family-gotham-xnarrow | Arial Narrow, sans-serif |
| Gotham Condensed | --font-family-gotham-condensed | Arial Narrow, sans-serif |
| Gotham Narrow | --font-family-gotham-narrow | Arial, sans-serif |

## Font Weights

| Token | Wert | Verwendung |
|-------|------|------------|
| --font-weight-regular | 400 | Fließtext |
| --font-weight-medium | 500 | Betonte Elemente |
| --font-weight-bold | 700 | Headlines, CTAs |
| --font-weight-black | 900 | Display Headlines |

### Body Styles

| Style | Größe | Line Height |
|-------|-------|-------------|
| .body-lg | 18px | 28px |

| .body-xs | 12px | 16px |

## Usage

```html
<h1 class="display-1">Breaking News</h1>
<p class="body-md">Standard Fließtext...</p>
```

```css
.custom-headline {
  font-family: var(--font-family-gotham-xnarrow);
  font-weight: var(--font-weight-bold);
}
```
