# Colors

Das BILD Design System verwendet eine **4-Layer Token Architektur** für Farben:

1. **Primitives** - Rohe Farbwerte (Layer 0)
2. **Brand Mapping** - Markenspezifische Zuordnung (Layer 1)
3. **Semantic** - Bedeutungsbasierte Tokens (Layer 2)
4. **Component** - Komponenten-spezifische Tokens (Layer 3)

---

## Semantic Colors

Bedeutungsbasierte Tokens - reagieren auf **Light/Dark Mode**.

> **Tipp:** Wechsle das Theme im Toolbar um Light/Dark Mode zu sehen.

### Text Colors

| Token | Light | Dark | Verwendung |
|-------|-------|------|------------|
| `--text-color-primary` | #232629 | #F2F4F5 | Haupttext |
| `--text-color-secondary` | #666B70 | #AEB4B9 | Sekundärtext |
| `--text-color-accent` | #DD0000 | #F52D2D | Akzenttext |
| `--text-color-inverse` | #FFFFFF | #232629 | Invertierter Text |

### Surface Colors

| Token | Light | Dark | Verwendung |
|-------|-------|------|------------|
| `--surface-color-primary` | #FFFFFF | #232629 | Hauptfläche |
| `--surface-color-secondary` | #F2F4F5 | #1C1C1C | Sekundärfläche |
| `--surface-color-canvas` | #F0F2F4 | #141414 | Hintergrund |

### Core Colors

| Token | Light | Dark | Verwendung |
|-------|-------|------|------------|
| `--core-color-primary` | #DD0000 | #DD0000 | Primäre Markenfarbe |
| `--core-color-secondary` | #232629 | #F2F4F5 | Sekundäre Farbe |

## Brand Colors

Die **Dual-Axis Architektur** trennt Farben nach Marke:

### data-color-brand="bild"
- Primär: **BILD Rot** (#DD0000)
- Für: BILD App, BILD.de

### data-color-brand="sportbild"
- Primär: **SportBILD Blau** (#0066B3)
- Für: SportBILD App, sportbild.de

```html
<!-- BILD Farben -->
<html data-color-brand="bild" data-theme="light">

<!-- SportBILD Farben -->
<html data-color-brand="sportbild" data-theme="dark">
```

---

## Usage

```css
/* Semantic Tokens verwenden */
.my-component {
  color: var(--text-color-primary);
  background: var(--surface-color-primary);
  border-color: var(--border-color-medium-contrast);
}

/* Akzentfarbe */
.highlight {
  color: var(--text-color-accent);
}
```
