# Typography

Das BILD Design System verwendet **Gotham** als primäre Schriftfamilie mit verschiedenen Breiten und Gewichten.

---

## Font Families

### Gotham XNarrow

Die engste Variante - für Headlines und kompakte Darstellungen.

<FontFamilyItem
  name="Gotham XNarrow"
  value="'Gotham XNarrow', 'Arial Narrow', sans-serif"
  cssVar="--font-family-gotham-xnarrow"
/>

### Gotham Condensed

Kondensierte Variante - für Subheadlines und UI-Elemente.

<FontFamilyItem
  name="Gotham Condensed"
  value="'Gotham Condensed', 'Arial Narrow', sans-serif"
  cssVar="--font-family-gotham-condensed"
/>

### Gotham Narrow

Standard-Variante für Fließtext.

<FontFamilyItem
  name="Gotham Narrow"
  value="'Gotham Narrow', Arial, sans-serif"
  cssVar="--font-family-gotham-narrow"
/>

---

## Font Weights

| Token | Wert | Verwendung |
|-------|------|------------|
| `--font-weight-regular` | 400 | Fließtext |
| `--font-weight-medium` | 500 | Betonte Elemente |
| `--font-weight-bold` | 700 | Headlines, CTAs |
| `--font-weight-black` | 900 | Display Headlines |

---

## Type Scale

Das Design System verwendet eine responsive Typografie-Skala.

### Display Styles

| Style | xs | sm | md | lg |
|-------|----|----|----|----|
| `.display-1` | 40px | 48px | 56px | 72px |
| `.display-2` | 32px | 40px | 48px | 56px |

### Heading Styles

| Style | xs | sm | md | lg |
|-------|----|----|----|----|
| `.heading-1` | 28px | 32px | 40px | 48px |
| `.heading-2` | 24px | 28px | 32px | 40px |
| `.heading-3` | 20px | 24px | 28px | 32px |
| `.heading-4` | 18px | 20px | 24px | 28px |

### Body Styles

| Style | Größe | Line Height | Verwendung |
|-------|-------|-------------|------------|
| `.body-lg` | 18px | 28px | Lead Text |
| `.body-md` | 16px | 24px | Standard Body |
| `.body-sm` | 14px | 20px | Captions, Labels |
| `.body-xs` | 12px | 16px | Fine Print |

---

## Typography Classes

Die Typography-Klassen gruppieren alle relevanten Eigenschaften:

```css
.display-1 {
  font-family: var(--font-family-gotham-xnarrow);
  font-weight: var(--font-weight-black);
  font-size: var(--display-1-font-size);
  line-height: var(--display-1-line-height);
  letter-spacing: var(--display-1-letter-spacing);
}
```

---

## Usage

```html
<!-- Headlines -->
<h1 class="display-1">Breaking News</h1>
<h2 class="heading-1">Artikel Headline</h2>

<!-- Body Text -->
<p class="body-md">Standard Fließtext...</p>
<span class="body-sm">Caption oder Label</span>
```

```css
/* Oder via CSS Custom Properties */
.custom-headline {
  font-family: var(--font-family-gotham-xnarrow);
  font-size: var(--heading-1-font-size);
  font-weight: var(--font-weight-bold);
  line-height: var(--heading-1-line-height);
}
```

## Responsive Behavior

Die Typografie passt sich automatisch an Breakpoints an:

| Breakpoint | Min-Width | Skalierung |
|------------|-----------|------------|
| `xs` | 320px | Basis (Mobile) |
| `sm` | 390px | +10-15% |
| `md` | 600px | +20-30% |
| `lg` | 1024px | +40-60% |

> **Tipp:** Responsive Größen werden automatisch via `@media` Queries angewendet - kein JavaScript erforderlich.
